# Activity Skill Examples

## 示例 1：基于官网链接的申请

**输入**：
> 我要申请“行业数字化转型示范案例”，这是活动链接：https://example.com/awards2026/rules，这是我们的项目介绍文档 @project_intro.docx

**Agent 响应**：
1. **分析规则**（从链接）：
   - 截止日期：2026年5月30日
   - 评分标准：创新性(30%)，实施难度(20%)，社会效益(30%)，推广价值(20%)。
   - 必需材料：申请表、项目报告（不超过5000字）、第三方检测报告。
2. **分析资料**（从文档）：
   - 项目：某智慧工厂改造。
   - 亮点：生产效率提升40%，能耗降低20%。
3. **生成草稿**：
   - 生成一份符合上述评分标准的项目报告大纲及核心内容草稿。

**输入**：
> 帮我写一个“优秀员工奖”的申请。要求突出团队协作和业绩贡献。我去年的业绩是销售额1000万，是团队第一，还带了2个新人。

> # 优秀员工奖申请书
>
> ## 申请理由
>
> ### 1. 卓越的业绩贡献
> 在过去的一年中，我始终以高标准要求自己，最终实现了 **1000万元** 的个人销售额，位列**团队第一**。这一成绩不仅超额完成了公司下达的 KPI，也为部门的整体业绩增长做出了核心贡献。
>
> ### 2. 团队协作与人才培养
> 除了个人业务，我也深知团队力量的重要性。在过去一年中，我主动担任了**2名新员工**的导师（Mentor），通过通过“传帮带”的方式，帮助他们快速适应岗位并产出业绩。
